# Transaction Data Analysis Dashboard

## Overview

This project was developed for learning purposes to gain hands-on experience with the end-to-end Power BI development process, including data preparation, visualization, interactivity, and report deployment.

The dataset used in this project is non-confidential and was utilized solely for educational and practice purposes.

---

## Objectives

* Understand the complete Power BI workflow
* Practice data cleaning and transformation techniques
* Build interactive dashboards and reports
* Learn report navigation and user experience features
* Explore publishing and sharing reports using Power BI Service

---

## Features Implemented

### Data Preparation

* Loaded data from Excel into Power BI
* Performed data profiling
* Corrected data types
* Split and transformed columns using Power Query

### Dashboard Development

* Designed and formatted slicers
* Adjusted visual size and positioning
* Added an Age Group column
* Created line charts and matrix visuals
* Applied conditional formatting
* Synchronized slicers across report pages

### User Experience

* Created bookmarks for Transactions view
* Created bookmarks for Remaining Balance view
* Improved report navigation and interactivity

### Deployment

* Published the report to Power BI Service
